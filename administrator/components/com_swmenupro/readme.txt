************* Install *************
Component Installation
----------------------
In Administrator Menu, go to Components, Install/Uninstall...
Go to the section Upload new component, browse and select the com_swmenupro8.0_beta.zip file...
Press the Upload File button.
The component will automatically install a module when it installs.
